Codeforces 1213 Problem G

Author: Yojahuang Date: 八月 31, 2019  22:25:13 Category: 競程

給你一棵有$n$個點的樹（$1 \leq n \leq 3 \cdot 10^5$），每條邊都有一個權重$w$
現在有Q筆詢問（$1 \leq Q \leq 3 \cdot 10^5$），每筆詢問有一個值$Q_i$
問你有多少個點對$(u, v)(u < v)$滿足兩點路徑上最大權重不超過$Q_i$。

首先觀察到這題必須要使用Disjoint Set來解題。

先把每條邊依照權重大小做排序，再來把$u,v$在並查集作合併，並且記錄這次合併後答案會增加多少。
